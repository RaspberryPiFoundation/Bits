## Primary Button

```js
<Button>Click me!</Button>
```

### Variants

#### Secondary Button

```js
<Button secondary>Secondary button</Button>
```

### Light-on-Dark

```js
<LightOnDark uBgcolor="ui-black">
  <Button secondary lightOnDark>Use this one on dark backgrounds</Button>
</LightOnDark>
```

## Block

```js
<Button block>Block Button</Button>
```
